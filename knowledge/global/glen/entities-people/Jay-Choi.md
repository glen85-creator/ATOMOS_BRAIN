# Jay Choi

## 개요

"인디해커 라이프" 채널을 운영하는 한국어 YouTube 크리에이터. 인디 해킹·AI 도구 활용 콘텐츠를 다룸. 안드레이 카파시의 LLM Wiki 아이디어를 "나 자신"을 주제로 적용하는 튜토리얼 영상(2026-04-18)으로 본 볼트의 구체적 구현 패턴을 제시했다.

## 역할/소속

- YouTube 채널 "인디해커 라이프" 운영
- 인디 해커 (개인 서비스 제작·실패·인사이트 공유)

## 주요 의사결정·기여

- **나 자신 = 위키 주제**: 카파시·일반 사례는 외부 지식(논문·기사) 정리에 LLM Wiki를 쓰지만, Jay는 "찾아도 안 나오는 정보는 나에 대한 정보"라는 관점에서 자기 자신을 위키화하는 방향을 제시.
- **구현 스택 추천**: Obsidian + Claude Code + obsidian-git + Obsidian Web Clipper 조합. 본 볼트가 그대로 따른다.
- **초기 시드 전략**: Claude의 기존 대화 메모리를 활용해 자기 자신에 대한 "백과사전"을 생성해 raw에 시드.
