# 요기요

## 개요

한국 배달 플랫폼. [[global/glen/entities-organizations/HBS]]의 배달 매출 채널 중 하나. HBS Dashboard와의 통합은 2026-04-17 handoff 문서로 정리됨 (2026-04-17-hbs-yogiyo-integration-handoff).

## 노트

- 운영사: GS리테일이 2021년 인수 (Delivery Hero 한국 사업 분리). 시장 점유율은 [[global/glen/entities-organizations/배달의민족]]에 이은 2~3위권.
- HBS Dashboard 통합은 하이픈 API 경유. `delivery_orders`·`delivery_settlements`·`delivery_reviews` 테이블에 적재.
- 배달앱 3사([[global/glen/entities-organizations/배달의민족]]·[[global/glen/entities-organizations/쿠팡이츠]]·요기요) 중 통합 우선순위·이슈는 handoff 요약에 정리.
