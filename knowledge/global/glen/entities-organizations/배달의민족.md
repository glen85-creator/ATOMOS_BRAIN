# 배달의민족

## 개요

한국 1위 배달 플랫폼. [[global/glen/entities-organizations/HBS]]의 배달 매출의 주요 채널 중 하나. 하이픈 API로 매출·정산·리뷰 데이터 수집. HBS Dashboard의 `delivery_orders`, `delivery_settlements`, `delivery_reviews` 테이블 원천.

## 노트

- 운영사: 우아한형제들 (Woowa Brothers)
- HBS 배달 손익 분석 페이지(`/delivery`, `/delivery/analytics`)의 주요 데이터 소스
