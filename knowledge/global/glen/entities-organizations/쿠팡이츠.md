# 쿠팡이츠

## 개요

쿠팡이 운영하는 배달 플랫폼. [[global/glen/entities-organizations/HBS]]의 배달 매출 채널. 하이픈 API로 매출·정산·리뷰 데이터 수집.

## 노트

- 운영사: 쿠팡 (Coupang Inc., NYSE: CPNG). 2019년 출범, 2021년부터 단건 배달 모델로 점유율 급성장.
- HBS Dashboard 통합 패턴은 [[global/glen/entities-organizations/배달의민족]]·[[global/glen/entities-organizations/요기요]]와 동일 (`delivery_*` 테이블 + 하이픈 API).
- 단건 배달 모델 때문에 정산 데이터의 시간 단위·수수료 구조가 다른 두 플랫폼과 다름 — 통합 시 정규화 로직 필요.
